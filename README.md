# wechat-mp-layout · 公众号排版提示词 / 技能（四风格）

把写好的文章一键排成**可直接粘贴进微信公众号编辑器的全内联 HTML**。
两种用法，零代码：复制 `PROMPT.md` 给任何 AI 直接用，或安装 `公众号排版.skill` 当成 Claude / Cowork 技能。

![演示：纯文本草稿 → 一键排成可粘进公众号的样式](assets/demo.gif)

## 为什么需要它

公众号编辑器会**删掉** `<style>` 标签、class、外链 CSS——网上大多数 AI 排版产出粘进去就乱。
本提示词强制 AI 只输出**全内联样式**的 HTML，并内置四套经过实战验证的视觉风格：

| 风格 | 适用内容 | 调性 |
|------|---------|------|
| A · 温柔高级 | 生活 / 情感 / 自我关怀 / 种草 | 暖棕裸粉，治愈留白 |
| B · 科技信息感 | AI / 科技 / 财经 / 教程（数据感强） | 深蓝荧光青，理性清晰 |
| C · 黑金女王感 | 认知 / 观点 / 主理人独白（锋利） | 黑金象牙白，锋利高级 |
| D · 莫兰迪扁平 | 知识科普 / 方法论 / 品牌表达（温和） | 莫兰迪低饱和，扁平克制 |

AI 会先按文章主题**自动判定风格**并说明理由，你可随时改判。

## 两种用法

**用法一 · 当提示词（任何 AI 都能用，30 秒）**

1. 打开 `PROMPT.md`，全文复制，粘给你的 AI
2. 接着粘你的文章正文（可指定风格 A/B/C/D，不指定就自动判）
3. AI 输出完整 HTML → 存成 `.html` → 浏览器打开 → 全选（Ctrl/Cmd+A）复制 → 粘进公众号后台编辑器
4. 图片占位框在编辑器里手动换成真图（公众号要求用自己的图床）

**用法二 · 装成技能（Claude / Cowork）**

下载 `公众号排版.skill`，在客户端点「保存技能 / 安装」即可。装好后直接说"公众号排版"+丢文章就触发。技能内容就是 `SKILL.md`，与 `PROMPT.md` 同源。

## 内置规则（踩坑沉淀）

- 全内联样式，无 class/id/`<style>`/外链 —— 编辑器粘贴不丢样式
- 手机端优先：宽度 100%，正文 ≥15px
- 一篇只用 1 主色 + 1 点缀色 + 中性色，不堆色
- 正文不用纯黑 `#000`（显廉价），用深灰
- 输出带 `<meta charset="utf-8">` 的完整文档 —— 本地打开不乱码
